# Bag F-Droid Repository

Add this repository to F-Droid or Obtainium:

**Repository URL**: `https://uxavia.github.io/fdroid-repo`

## How to add in F-Droid
1. Open F-Droid → Settings → Repositories → Add repository
2. Enter: `https://uxavia.github.io/fdroid-repo`

## How to add in Obtainium
1. Add App → enter: `https://github.com/UXAVIA/bag`
